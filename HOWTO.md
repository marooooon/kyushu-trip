# 📖 しおりサイトの使い方ガイド

## ブラウザでしおりを確認する

### ローカルサーバーを起動する（推奨）

```bash
# プロジェクトディレクトリで実行
npm run dev
```

- ブラウザが自動で開き `http://localhost:3000` が表示されます
- ファイルを編集して保存すると**ページが自動リロード**されます
- `Ctrl+C` でサーバーを停止できます

### live-server が初回インストールされる場合

`npx live-server` は初回実行時に自動でインストールされます（数秒かかります）。

### 代替方法

```bash
# Python を使う場合（インストール不要）
python3 -m http.server 3000
# → http://localhost:3000 をブラウザで手動で開く

# serve を使う場合
npm run preview
```

## しおりに情報を追加する

### Step 1: MEMO.md に書く

`MEMO.md` を開いて、追加したい情報を `###` 見出しで記入します。

```markdown
## 🗺️ 行きたい場所・観光スポット

### 指宿・砂むし温泉
- **県**: 鹿児島県
- **コメント**: 砂に埋まる日本唯一の温泉体験！
- **犬連れ**: 車待機（温泉入浴中）
- **参考URL**: https://www.ibusuki.or.jp
```

**書けるセクション:**

| セクション | 書く場所 | 反映先 |
|-----------|---------|--------|
| 🚐 レンタルショップ | `## 🚐 キャンピングカーレンタルショップ` | `#rental` |
| 🗺️ 観光スポット | `## 🗺️ 行きたい場所・観光スポット` | `#itinerary` |
| 🍽️ グルメ | `## 🍽️ 食べたいもの・グルメ` | `#gourmet` |
| ♨️ 温泉 | `## ♨️ 行きたい温泉` | 新規温泉セクション |

### Step 2: Claude Code でスキルを実行する

```
/trip-update
```

MEMO.md を読み込んで `index.html` の該当セクションに自動追加されます。

### ルール

- `### （例）` で始まるエントリはサンプルなのでスキップされます
- 反映済みエントリには `<!-- 反映済 -->` タグが自動で付きます（重複防止）
- 既存のHTML・CSS・JSは変更されません（追加のみ）

## ファイルを直接編集する

HTML を直接編集したい場合:

| ファイル | 編集内容 |
|---------|---------|
| `index.html` | ページ構造・コンテンツ |
| `css/style.css` | デザイン・レイアウト |
| `js/main.js` | インタラクション（スクロール・チェックリスト） |

編集後は `npm run dev` で起動中なら自動でブラウザに反映されます。

---

## チェックリストの使い方

「🐾 コロの持ち物チェックリスト」のチェックボックスは**ブラウザのLocalStorageに保存**されます。  
ページを閉じてまた開いてもチェック状態が維持されます。

## よくある質問

**Q: `npm run dev` でエラーが出る**  
A: Node.js がインストールされているか確認してください。`node -v` でバージョンが表示されれば OK です。

**Q: ブラウザが自動で開かない**  
A: `http://localhost:3000` を手動でブラウザのアドレスバーに入力してください。

**Q: MEMO.md を更新したのにしおりに反映されない**  
A: `### （例）` が付いていないか、`<!-- 反映済 -->` が付いていないか確認してください。その後 `/trip-update` を再実行してください。
